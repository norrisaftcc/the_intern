    zsh: command not found: ls

    uh oh, i think i blew up my .zshrc. now i know that i can make a new one, but how do i do that without being able to use my terminal

    Here are two ways to create a new .zshrc without using terminal commands:

    1. Use a text editor GUI (like TextEdit, Sublime, VS Code) to create a file named ".zshrc" in your home directory (~/)
    2. Open Finder, press Cmd+Shift+G, type "~/", create a new file there and name it ".zshrc"

    For basic functionality, add these lines to your new .zshrc:

    bash
    export PATH=/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin
alias ls='ls -G'

    After creating the file, restart your terminal.
